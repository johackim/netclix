DEPRECATED : vodlocker.to, the web service behind netclix is closed.

[![asciicast](https://asciinema.org/a/125918.png)](https://asciinema.org/a/125918?t=0:01)

## Prerequisites

- [Node.js](https://nodejs.org/) v6.0+

## Installation

```bash
npm install --global netclix
```

or

```bash
# Arch Linux
pacaur -S netclix
```

## Usage

```bash
netclix
```

## Running the tests

```bash
make test
```
